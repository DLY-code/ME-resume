# 机器学习回归模型面试问答总结

## 1. 这些模型是怎么使用的（pytorch搭建的吗？）

- 我在温度预测任务中主要使用了 XGBoost、LightGBM、CatBoost 和 Random Forest 这几类集成学习模型。
- 这些模型并不是用 PyTorch 搭建的，而是直接调用各自的官方 Python 库（如 xgboost、lightgbm、catboost、sklearn.ensemble.RandomForestRegressor）进行训练和预测。
- 典型流程是：
  1. 数据预处理（如缺失值填充、特征工程、归一化等）；
  2. 划分训练集和测试集；
  3. 用对应库的 API 初始化模型，设置参数；
  4. 调用 fit 方法训练，predict 方法预测；
  5. 评估模型效果（如 MAE、RMSE、R2 等）。
- 这些模型属于树模型（集成方法），不需要像深度学习那样用 PyTorch 手动搭建网络结构。

## 2. 有什么参数，怎么选择参数？

- 主要参数有：
  - n_estimators / num_boost_round：树的数量，越多模型越复杂，容易过拟合；
  - max_depth：单棵树的最大深度，控制模型复杂度；
  - learning_rate（对提升树有效）：步长，越小训练越慢但泛化更好；
  - subsample / colsample_bytree：采样比例，防止过拟合；
  - reg_alpha / reg_lambda：正则化参数，抑制过拟合；
  - random_state / seed：随机种子，保证结果可复现。
- 参数选择方法：
  - 通常先用默认参数训练，观察效果；
  - 关键参数（如 n_estimators、max_depth、learning_rate）用网格搜索（GridSearchCV）、随机搜索（RandomizedSearchCV）或贝叶斯优化等方法调优；
  - 也会结合交叉验证（如 KFold）评估参数组合的泛化能力。

## 3. 模型的优劣

- XGBoost：
  - 优点：速度快、效果好、支持缺失值、特征重要性解释强、调参灵活。
  - 缺点：参数较多，调参复杂，对异常值敏感。
- LightGBM：
  - 优点：训练速度更快，内存占用低，支持大规模数据，自动类别特征处理。
  - 缺点：对小数据集有时不如 XGBoost，默认参数下可能过拟合。
- CatBoost：
  - 优点：对类别特征支持最好，调参简单，鲁棒性强，收敛快。
  - 缺点：训练速度略慢于 LightGBM，生态不如 XGBoost 丰富。
- Random Forest：
  - 优点：原理简单，抗过拟合能力强，对参数不敏感，易于并行。
  - 缺点：预测速度慢，模型解释性较弱，效果有时不如提升树。
