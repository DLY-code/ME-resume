# PyTorch、NumPy、Pandas 简明面试整理

这份笔记主要聚焦应用层面，帮助你在面试中清楚说明这三个库分别是做什么的、平时怎么用、常被问什么。

## 1. 先用一句话分清三者

- `NumPy`：做数值计算和矩阵运算；`Pandas`：做表格数据处理和分析；`PyTorch`：做深度学习建模、训练和推理。

如果面试官让你快速介绍，可以这样说：

> `NumPy` 更偏底层数值计算，`Pandas` 更偏结构化数据处理，`PyTorch` 更偏模型训练和深度学习应用。

---

## 2. `NumPy`

### 2.1 它的作用

`NumPy` 是 Python 里最常用的数值计算库，核心对象是 `ndarray`，也就是多维数组。它适合做：向量和矩阵运算、数组计算、线性代数、机器学习中的数据表示和基础计算。

通俗理解：如果你要处理大量数字，并且希望计算快、写法简洁，一般会用 `NumPy`。

### 2.2 常用功能

- 创建数组：`array()`、`zeros()`、`ones()`；查看形状：`shape`；改变形状：`reshape()`；维度变换：`transpose()`。
- 数组运算：加减乘除；聚合统计：`sum()`、`mean()`、`max()`、`min()`；条件筛选；广播机制。

### 2.3 面试中怎么描述它

> `NumPy` 主要用于高效数值计算，尤其适合矩阵和数组操作。在机器学习项目中，经常用它做数据表示、特征处理和基础数学运算。

### 2.4 常见面试问题

#### 1. `NumPy` 是做什么的？

> `NumPy` 是 Python 的数值计算基础库，核心是多维数组，主要用来做高效的数组和矩阵运算。

#### 2. `NumPy` 和 Python `list` 有什么区别？

> `list` 更通用，`NumPy` 更适合数值计算。`NumPy` 支持批量运算，处理大量数字时效率更高。

#### 3. 你在项目里一般怎么用 `NumPy`？

> 我通常用它做数据转换、数组处理、统计计算，以及为后续模型输入准备数值数据。

---

## 3. `Pandas`

### 3.1 它的作用

`Pandas` 是 Python 里最常用的数据分析库，特别适合处理表格数据。它的核心对象是 `Series` 和 `DataFrame`，适合做：读取 CSV、Excel，数据清洗，缺失值处理，条件筛选，分组统计，多表合并。

通俗理解：如果数据长得像 Excel 表格，那通常就很适合用 `Pandas`。

### 3.2 常用功能

- 读写数据：`read_csv()`、`read_excel()`；查看数据：`head()`、`info()`、`describe()`；选取行列：`loc`、`iloc`。
- 过滤数据、排序 `sort_values()`、去重 `drop_duplicates()`、缺失值处理 `isna()` `dropna()` `fillna()`、分组统计 `groupby()`、表连接 `merge()`。

### 3.3 面试中怎么描述它

> `Pandas` 主要用于结构化数据处理，尤其适合做数据清洗、分析和预处理。在实际项目里，经常先用 `Pandas` 处理原始数据，再送到模型里。

### 3.4 常见面试问题

#### 1. `Pandas` 是做什么的？

> `Pandas` 是处理表格数据和数据分析的常用库，适合做数据清洗、筛选、统计和合并。

#### 2. `Pandas` 和 `NumPy` 的区别是什么？

> `NumPy` 更偏数值计算，`Pandas` 更偏表格数据处理。`Pandas` 底层很多能力也是建立在 `NumPy` 之上的。

#### 3. 你常用哪些 `Pandas` 功能？

> 我常用读文件、筛选列、缺失值处理、分组统计、排序和多表合并这些操作。

#### 4. `loc` 和 `iloc` 有什么区别？

> `loc` 按标签取数据，`iloc` 按位置下标取数据。

---

## 4. `PyTorch`

### 4.1 它的作用

`PyTorch` 是一个深度学习框架，核心对象是 `Tensor`。它适合做：搭建神经网络、训练模型、做推理预测、使用 GPU 加速计算。

通俗理解：如果 `NumPy` 是做普通数值计算，那么 `PyTorch` 更像是“专门用来训练模型的计算工具”。

### 4.2 常用功能

- 张量创建与运算：`tensor()`；模型搭建：`nn.Module`；常见网络层：全连接层、卷积层、激活函数；损失函数；优化器：`SGD`、`Adam`。
- 数据加载：`Dataset`、`DataLoader`；模型训练和验证；模型保存与加载。

### 4.3 面试中怎么描述它

> `PyTorch` 主要用于深度学习任务，比如图像分类、目标检测、文本分类等。我一般用它来定义模型、加载数据、训练参数，并完成验证和推理。

### 4.4 常见面试问题

#### 1. `PyTorch` 是做什么的？

> `PyTorch` 是一个深度学习框架，主要用来搭建神经网络、训练模型和进行推理。

#### 2. `PyTorch` 和 `NumPy` 的区别是什么？

> 两者都能做数组运算，但 `PyTorch` 更适合深度学习，因为它支持模型训练、自动求导和 GPU 加速。

#### 3. 你平时怎么用 `PyTorch`？

> 我一般用它定义模型、准备数据加载流程、设置损失函数和优化器，然后完成训练、验证和预测。

#### 4. `Dataset` 和 `DataLoader` 是什么？

> `Dataset` 定义数据怎么读取，`DataLoader` 负责按批次加载、打乱顺序和喂给模型。

#### 5. `model.train()` 和 `model.eval()` 的作用是什么？

> `train()` 用于训练模式，`eval()` 用于验证或推理模式，确保模型在不同阶段使用合适的行为。

---

## 5. 三者在实际项目中的关系

很多项目里，这三个库会配合使用。一个常见流程是：先用 `Pandas` 读取和清洗数据，再用 `NumPy` 做数值转换和基础计算，最后用 `PyTorch` 把数据送进模型进行训练或预测。

面试里可以这样回答：

> 在实际项目中，我通常先用 `Pandas` 做数据预处理，再把数据转成 `NumPy` 数组或 `PyTorch Tensor`，最后用 `PyTorch` 完成模型训练和推理。

---

## 6. 适合面试的简短回答

如果面试官问：“你了解 `PyTorch`、`NumPy`、`Pandas` 吗？”

> 了解。`NumPy` 主要做高效数值计算和矩阵运算，`Pandas` 主要做表格数据处理和分析，`PyTorch` 主要做深度学习建模和训练。实际项目里，我一般会先用 `Pandas` 做数据清洗，再转换成 `NumPy` 或 `PyTorch` 需要的格式，最后用于模型训练或预测。
