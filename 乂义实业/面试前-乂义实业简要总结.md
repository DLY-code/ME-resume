# 乂义实业面试前简要总结

资料来源：`乂义实业简介_2026版_V1.1.pdf`

## 1. 公司一句话

乂义实业是一家做 **精密光电系统 + 计算光学系统** 的公司，核心市场集中在 **航空航天、科研、军工、国防** 场景。

## 2. 我现在对这家公司的核心理解

- 成立时间：`2016` 年
- 注册资本：`1376 万`
- 核心方向：**计算光学、微扫描、超分辨、精密扫描/调制系统**
- 核心客户/市场：**航天、航空、军工、科研**
- 公司标签：**高新技术 / 专精特新 / 科技型**
- 融资信息：材料里提到有 `Pre-A`、`Pre-A+`

## 3. 这家公司在做什么

从材料看，公司不是做通用互联网产品，也不是纯算法公司，它更偏：

- 精密光电系统设计
- 计算光学系统开发
- 微扫描超分辨技术工程化
- 嵌入式实时重构
- 高可靠交付和批量制造

简单说，它更像一家 **“高端光电 + 计算成像 + 工程化落地”** 的技术公司。

## 4. 他们强调的核心技术

公司材料里反复强调的是：

- 基于 **光信号主动精密调制（微扫描）** 的计算光学技术
- **超精密信息调制**
- **光学全链路成像物理建模**
- **嵌入式实时数据重构**
- **超分辨成像/分辨率增强**

关键词可以记成：

`微扫描`、`超分辨`、`计算光学`、`精密调制`、`嵌入式实时重构`

## 5. 他们的产品/成果给人的感觉

材料里出现的产品和成果，整体给人的感觉是：

- 不是停留在论文或算法验证
- 而是已经在做 **工程化、产品化、批量交付**
- 而且应用在 **航天/机载/特种光电系统** 这种高要求场景

材料里明确提到的亮点包括：

- 十余个系统发明专利
- 国内首个空间高速微扫超分辨项目相关表述
- 与航天院所、中科院相关单位合作
- 强调商业航天背景下的 **快速集成、批量制造、高可靠性**

## 6. 面试时你最该注意的点

### 6.1 不要把它当成普通 AI 公司

这家公司虽然有“计算”和“重构”成分，但核心底色是：

- 光学
- 光电系统
- 精密工程
- 航天/军工类项目

所以面试里不要一直往“大模型、通用 CV、互联网 AI”上带。  
更好的切入词是：

- 工程化落地
- 复杂系统集成
- 嵌入式/边缘部署
- 高可靠性
- 实时性
- 物理建模

### 6.2 他们很可能更看重“工程可信度”

从材料看，这家公司很强调：

- 高精度
- 高可靠
- 实时
- 批量交付
- 自主可控

所以你回答问题时，尽量多讲：

- 你怎么把模型/系统跑通
- 怎么处理部署和集成问题
- 怎么保证结果稳定
- 怎么适应真实场景而不是只做 demo

### 6.3 如果你讲自己的项目，优先往这些方向靠

- 检测/跟踪系统的工程化集成
- 边缘设备部署
- 算法从训练到交付的闭环
- 数据质量、精度、鲁棒性、实时性
- 和硬件、设备、场景约束打交道的经验

### 6.4 可能会比较看重保密意识和表达边界

因为客户背景偏航天/军工/科研，面试交流里建议：

- 说话稳一点
- 不夸大自己不懂的东西
- 对“精度、指标、可靠性”这类词保持严谨

## 7. 你可以准备的自我贴合点

如果你要把自己的经历往这家公司贴，比较合适的角度是：

- 我做过检测、跟踪、ReID 和部署，不只是训练模型，也做过系统落地
- 我做过边缘端/NPU 部署，理解资源受限场景下的工程约束
- 我对“模型怎么真正跑起来、稳定起来、交付出去”更有感觉
- 我愿意补光学/成像/硬件相关知识，能接受偏工程和交叉学科的工作方式

- 这是一家做 **精密光电 + 计算光学 + 微扫描超分辨** 的公司
- 市场偏 **航天、航空、军工、科研**
- 关键词：`微扫描`、`超分辨`、`计算光学`、`嵌入式实时重构`、`高可靠交付`
- 面试时多讲 **工程化、部署、实时性、稳定性、系统集成**
- 少讲空泛 AI 热词，多讲你怎么把项目真正落地
